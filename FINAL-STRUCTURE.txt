================================================================================
INSTALLATION WORKFLOW
================================================================================

1. Clone Repository
   $ git clone https://github.com/sparkry/claude-skills-marketplace.git
   $ cd claude-skills-marketplace

2. Navigate to Plugin
   $ cd plugins/research-workflow

3. Review Documentation
   $ cat README.md
   $ cat MANIFEST.md

4. Copy to Local Setup
   $ cp -r . ~/.claude/plugins/research-workflow

5. Configure (if needed)
   $ export SERPAPI_KEY=your_key
   $ export GOOGLE_DOCS_API_KEY=your_key

6. Use in Claude
   "Use the research-workflow plugin to analyze the SaaS market..."
