# presentation-extras

PowerPoint presentation and handout generation tools.

## Installation

```bash
/plugin marketplace add sparkst/sparkry-claude-skills
/plugin install presentation-extras@sparkry-claude-skills
```

## What's Included

**Skills:** powerpoint-generator

## Quick Reference

| Shortcut | Purpose |
|----------|---------|
| powerpoint-generator | Generate PPTX presentations and handouts from content |
