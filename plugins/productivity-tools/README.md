# productivity-tools

Excel file generation with conditional formatting, freeze panes, filters, and data validation.

## Installation

```bash
/plugin marketplace add sparkst/sparkry-claude-skills
/plugin install productivity-tools@sparkry-claude-skills
```

## What's Included

**Skills:** excel-generator

## Quick Reference

| Shortcut | Purpose |
|----------|---------|
| QEXCEL | Generate Excel files with formatting, filters, and validation |
