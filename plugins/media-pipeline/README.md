# media-pipeline

Video-to-transcript pipeline with audio extraction and WhisperX transcription.

## Installation

```bash
/plugin marketplace add sparkst/sparkry-claude-skills
/plugin install media-pipeline@sparkry-claude-skills
```

## What's Included

**Skills:** video-transcript

## Quick Reference

| Shortcut | Purpose |
|----------|---------|
| QTRANSCRIPT | Extract audio from video and transcribe with speaker diarization |
