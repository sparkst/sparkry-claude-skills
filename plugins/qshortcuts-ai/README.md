# QShortcuts AI

AI and architecture shortcuts for system design and prompt optimization.

## Installation

```bash
/plugin marketplace add sparkst/sparkry-claude-skills
/plugin install qshortcuts-ai@sparkry-claude-skills
```

## What's Included

**Skills:** QARCH, QPROMPT, QTRANSFORM

**Agents:** architecture-advisor, pe-designer

## Quick Reference

| Shortcut | Purpose |
|----------|---------|
| QARCH | Design learning AI systems (RAG, RLHF, personas) |
| QPROMPT | Optimize prompts for token efficiency |
| QTRANSFORM | Transform content for multiple platforms |

## Usage

```bash
QARCH: Design RAG system for 50K docs with feedback loops
QPROMPT: Optimize this prompt to <1000 tokens
QTRANSFORM: Transform for LinkedIn, Twitter, Email
```
