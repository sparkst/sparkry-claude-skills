# QShortcuts Support

Development support shortcuts for docs, git, research, and UX testing.

## Installation

```bash
/plugin marketplace add sparkst/sparkry-claude-skills
/plugin install qshortcuts-support@sparkry-claude-skills
```

## What's Included

**Skills:** QDOC, QGIT, QIDEA, QUX

**Agents:** docs-writer, release-manager, ux-tester

## Quick Reference

| Shortcut | Purpose |
|----------|---------|
| QDOC | Generate/update documentation |
| QGIT | Commit with conventional commits |
| QIDEA | Research and ideation (no code) |
| QUX | UX testing and accessibility check |

## Usage

```bash
QDOC                    # Update docs based on recent changes
QGIT                    # Commit with quality gates
QIDEA "OAuth providers" # Research without coding
QUX src/components/     # Generate UX test scenarios
```
