# QShortcuts Learning

Meta-learning and skill creation shortcuts.

## Installation

```bash
/plugin marketplace add sparkst/sparkry-claude-skills
/plugin install qshortcuts-learning@sparkry-claude-skills
```

## What's Included

**Skills:** QFEEDBACK, QLEARN, QCOMPACT, QSKILL

## Quick Reference

| Shortcut | Purpose |
|----------|---------|
| QFEEDBACK | Extract and integrate user feedback |
| QLEARN | Retrieve relevant learnings for current task |
| QCOMPACT | Consolidate large learning files |
| QSKILL | Create new agent+skill complex |

## Usage

```bash
QFEEDBACK           # Process feedback from codebase
QLEARN "error handling" --domain api
QCOMPACT learnings/testing/patterns.md
QSKILL "Test Reporter" --trigger QREPORT
```
