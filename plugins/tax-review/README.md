# tax-review

Tax return review and validation -- W-2, 1099, charitable deductions, ESPP.

## Installation

```bash
/plugin marketplace add sparkst/sparkry-claude-skills
/plugin install tax-review@sparkry-claude-skills
```

## What's Included

**Skills:** tax-review

## Quick Reference

| Shortcut | Purpose |
|----------|---------|
| tax-review | Review and validate tax returns against IRS rules and source documents |
