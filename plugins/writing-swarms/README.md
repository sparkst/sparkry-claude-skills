# writing-swarms

Multi-angle swarm generation for headlines and soundbites with cross-ranking.

## Installation

```bash
/plugin marketplace add sparkst/sparkry-claude-skills
/plugin install writing-swarms@sparkry-claude-skills
```

## What's Included

**Skills:** headline-swarm, soundbite-swarm

## Quick Reference

| Shortcut | Purpose |
|----------|---------|
| QHEADLINE | Generate click-worthy article headlines using 5-angle swarm |
| QSOUNDBITE | Generate memorable soundbites (2-7 words) using 5-angle swarm |
