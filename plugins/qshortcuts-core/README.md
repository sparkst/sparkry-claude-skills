# QShortcuts Core

Core TDD development shortcuts for Claude Code.

## Installation

```bash
/plugin marketplace add sparkst/sparkry-claude-skills
/plugin install qshortcuts-core@sparkry-claude-skills
```

## What's Included

**Skills:** QNEW, QPLAN, QCODET, QCODE, QCHECK, QCHECKF, QCHECKT

## Quick Reference

| Shortcut | Purpose |
|----------|---------|
| QNEW | Start new feature with requirements |
| QPLAN | Create implementation plan with SP estimates |
| QCODET | Write failing tests (TDD red phase) |
| QCODE | Implement to pass tests (TDD green phase) |
| QCHECK | Full quality review |
| QCHECKF | Fast quality check |
| QCHECKT | Test-only check |

## TDD Workflow

```
QNEW → QPLAN → QCODET → QCODE → QCHECK
```
