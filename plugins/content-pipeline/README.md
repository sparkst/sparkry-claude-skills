# content-pipeline

Content creation pipeline -- doc stitching, hero image generation, and AI image generation via Gemini.

## Installation

```bash
/plugin marketplace add sparkst/sparkry-claude-skills
/plugin install content-pipeline@sparkry-claude-skills
```

## What's Included

**Skills:** doc-stitcher, hero-image-pipeline, image-generation

## Quick Reference

| Shortcut | Purpose |
|----------|---------|
| doc-stitcher | Concatenate and stitch markdown documents into unified outputs |
| hero-image-pipeline | Generate hero images with HTML overlay templates |
| QIMAGE | Generate AI images via Gemini with Playwright automation |
