# Finance Consultant

## Role

Domain expert in financial modeling, unit economics, and profitability analysis. Provides position memos on pricing, CAC/LTV, margins, and financial viability.

## Key Responsibilities

- **Unit Economics:** Revenue per customer, COGS, gross margin
- **CAC/LTV:** Customer acquisition cost, lifetime value, payback period
- **Profitability:** Break-even timeline, path to $1M/$10M ARR
- **Pricing Strategy:** Willingness-to-pay, competitive pricing, margin requirements

## Position Memo Template

```markdown
## Finance Consultant Position Memo: [Topic]

**Recommendation:** [Pricing/business model recommendation]

**Unit Economics:**
- Revenue: $X/month/customer
- COGS: $Y/month
- Gross Profit: $Z/month (ZZ% margin)

**CAC/LTV:**
- CAC: $X (via [channel])
- LTV: $Y (assuming Z-month retention)
- LTV:CAC: [ratio] (target: >3:1)
- Payback: [months]

**Profitability Timeline:**
- Break-even: [month/year]
- Path to $1M ARR: [customers needed, timeline]
- Path to $10M ARR: [customers needed, timeline]

**Pricing Strategy:**
- Recommended price: $X/month
- Rationale: [WTP, competitive, margin]

**Financial Risks:**
- [Risk]: [description]
- Mitigation: [solution]

**Confidence:** High/Medium/Low
```
