# Changelog

All notable changes to the Strategy Workflow plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-28

### Added
- Initial release of Strategy Workflow plugin
- COS (Chief of Staff) agent for orchestrating strategic deliverables
- 5 specialist agents:
  - Strategic Advisor (market sizing, competitive positioning, GTM)
  - Finance Consultant (unit economics, CAC/LTV, profitability)
  - Product Manager (JTBD, market research, prioritization)
  - Legal Expert (compliance, data privacy, regulatory)
- 6 strategic skills:
  - Intake (request parsing and staffing)
  - PR-FAQ (Amazon Working Backwards methodology)
  - Buy-vs-Build (7-dimension decision matrix)
  - PMF Validation (JTBD, WTP, activation, retention)
  - Tenets (non-negotiable principles framework)
  - Security Review (risk-based third-party skill evaluation)
- Python tools:
  - TCO 3-year calculator (`tco_3y_calc.py`)
  - Decision tracker (`decision_tracker.py`)
- Comprehensive documentation and examples
- MIT License

### Features
- Intelligent specialist routing based on deliverable type
- Parallel specialist coordination
- Dissent management (preserves alternative viewpoints)
- Amazon-style quality standards
- Telemetry tracking
- Risk-based security review process
- Generic, business-agnostic framework
